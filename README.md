# Jasiri – Africa's Sports Community & Data Platform

A beautiful, fully functional SaaS landing page built with Next.js 14+, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting Started

### Prerequisites
- Node.js 18+
- npm, yarn, or pnpm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm run start
```

## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui (new-york style)
- **Icons**: Lucide React
- **Font**: System font stack (system-ui, -apple-system, BlinkMacSystemFont, Segoe UI)

## Page Sections

1. **Navbar** – Sticky navbar with mobile hamburger menu
2. **Hero** – Bold full-screen hero with CTA buttons
3. **Stats Bar** – Key platform statistics
4. **Features** – 6-feature grid with alternating black/white cards
5. **For Who** – 4 audience cards with hover effects
6. **How It Works** – 3-step process timeline
7. **Pricing** – 3-tier pricing (Free, AthletIQ, Imara)
8. **Testimonials** – 3 testimonials from African sports community
9. **CTA Banner** – Final conversion section
10. **Footer** – Full footer with links and social icons

## Design

- **Color Scheme**: Pure black, white, and grays only
- **Typography**: Inter with bold weights (800-900)
- **Responsive**: Mobile-first, fully responsive
